MFCUI 

MFCUI 大多是餘自繪的一些 MFC UI
為了不重複開發 餘將其整理到一起 作為 MFCUI庫

MFCUI 使用GDI+進行繪製 若要使用 需要自己調用 初始化GDI+代碼

結構 

class ui_base //ui_base.h
ui_base 是所有 MFCUI中 自繪UI 的基類 
其定義了基礎的 繪製功能

命名規則 

所有自繪的UI 其類名都只是在原MFC 類名後加上了 Draw(从CStatic派生的类除外) 
如 CButtonDraw

使用方法 

靜態創建
1	創建控件
2	關聯控件 MFCUI類
3	像MFCUI的基類一樣使用 MFCUI類

動態創建
1	定義MFCUI類實例
2	調用MFCUI類的 create方法
3	像MFCUI的基類一樣使用 MFCUI類
