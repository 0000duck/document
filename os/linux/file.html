檔案權限 目錄配置

檔案權限 

[dark@localhost ~]$ ls -l [--full-time]
drwxr-xr-x. 4	root	root	4096	3月 9 17:12	test
檔案權限 連接數	所有者	所在組	容量	修改日期	檔案名

d 目錄
- 檔案
| 連接檔(link file)
b 裝置檔案裡面的可供存儲的走遍設備
c 設備檔案裡面的串行端口 如鼠標 鍵盤等

rwx 讀 寫 執行
第一組rwx 所有者權限
第二組rwx 所有組權限
第三組rwx 其他用戶權限



	
權限對於檔案含義對於目錄含義權限分數

	
r可讀取檔案數據可查詢目錄下的檔案4

	
w可編輯檔案
(不可刪除 那需要目錄x權限)		新建 刪除 重命名
目錄下的 檔案		2

	
x執行進入目錄1

常用命令 

//改變a.text的所有組為 dark
chgrp dark a.text
//改變a連同旗下所有子孫目錄 檔案 所有組為 dark
chgrp -R dark a 



//改變 a.text 所有者 為 dark
chown dark a.text
chown -R dark a

//改變 a.text 所有者為 dark 所在組為 root
chown dark:root a.text 

//改變 所在組
chown .dark a.text
chown :dark a.text



//改變權限
chmod 644 a.text

		u	+(加入)	r
chmod	g	-(減去)	w
		o	=(設置)	x
		a

chmod u=rw,go=r a.text
chmod u+r a.text
chmod u-r a.text
chmod a-x a.text
