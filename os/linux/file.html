檔案權限 目錄配置

檔案權限 

[dark@localhost ~]$ ls -l [--full-time]
drwxr-xr-x. 4	root	root	4096	3月 9 17:12	test
檔案權限 連接數	所有者	所在組	容量	修改日期	檔案名

d 目錄
- 檔案
| 連接檔(link file)
b 裝置檔案裡面的可供存儲的走遍設備
c 設備檔案裡面的串行端口 如鼠標 鍵盤等

rwx 讀 寫 執行
第一組rwx 所有者權限
第二組rwx 所有組權限
第三組rwx 其他用戶權限



	
權限對於檔案含義對於目錄含義權限分數

	
r可讀取檔案數據可查詢目錄下的檔案4

	
w可編輯檔案
(不可刪除 那需要目錄x權限)		新建 刪除 重命名
目錄下的 檔案		2

	
x執行進入目錄1

常用命令 

//改變a.text的所有組為 dark
chgrp dark a.text
//改變a連同旗下所有子孫目錄 檔案 所有組為 dark
chgrp -R dark a 



//改變 a.text 所有者 為 dark
chown dark a.text
chown -R dark a

//改變 a.text 所有者為 dark 所在組為 root
chown dark:root a.text 

//改變 所在組
chown .dark a.text
chown :dark a.text



//改變權限
chmod 644 a.text

		u	+(加入)	r
chmod	g	-(減去)	w
		o	=(設置)	x
		a

chmod u=rw,go=r a.text
chmod u+r a.text
chmod u-r a.text
chmod a-x a.text

目錄配置 

/	(root 系統根目錄)

	/bin	可執行命令存放目錄

	/boot	內核文件
		/boot/grub	開機設置文件

	/dev	類似 /dev/null /dev/sda ...等設備文件

	/etc	各種軟體與系統配置文件
		/etc/init.d	各種軟體的啟動腳本

	/home	用戶主文件夾

	/lib	各種目標文件和函數庫
		/lib/modules	各種不同版本內核模塊 如硬件驅動

	/media	如cdroom 等暫存媒體
	/mnt	通常是臨時掛節點

	/opt	通常是第三方軟體安裝位置

	/proc	內存中的數據 如內核 軟體等

	/root	root 用戶主文件夾

	/sbin	開機或單用戶模式下 能執行的系統命令

	/srvc	網絡服務器提供的 數據放置處

	/sys	內存中的數據 如內核 文件系統等

	/tmp	臨時數據

	/usr	軟體默認安裝位置

/etc 

/etc/init.d	存放服務默認啟動腳本
/etc/X11	X Window相關配置

/usr 

/usr/X11R6	x window 系統重要數據
/usr/bin	大部分用戶可用命令
/usr/include	c/c++ header
/usr/lib	引用程序 函數庫 目標文件
/usr/local	系統管理員 在本機 安裝的軟體
/usr/sbin	非系統正常運行需要的命令
/usr/share	共享文件夾
/usr/src	源碼

/var 

/var/cache	應用程序緩存
/var/lib	應用程序執行中需要使用的數據 如 mysql為 /var/lib/mysql
/var/locl
/var/log
/var/mail
/var/run
/var/spool

注意 

os啟動時只會掛接 / 
故 /etc /bin /dev /lib /sbin 不能掛接到其它分區
