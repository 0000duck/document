shell script 參數讀取 函數

shell script 

linux 提供了 強大的 shell script 用於 簡化 重複的 shell 命令執行

example 

#!/bin/bash
#Program:
#       this program is test shell script
#History:
#       2013-03-21 illusive_man first release
#       2016-12-12 illusive_man second release
#Email:
#       zuiwuchang@gmail.com

echo "Hellow world"
exit 0


第1行 聲明了 使用bash shell
第[2,8]行 可選的聲明了 script 描述
echo 執行腳本 命令
exit 0 退出腳本 並設置 錯誤碼爲 0

參數讀取 

執行 script 時可以使用 參數 用 空格隔開
在script中 使用如下方式 操作 參數

$#		//獲取參數數量
$1 $2 ... $n	//操作第n個參數
$@		//獲取全部變量
$*		//獲取全部變量

參數讀取 

echo "參數總數  : $#"
echo "參數內容  : $@"
echo "參數內容  : $*"
echo "第n個參數 : $1"
echo

echo "遍歷參數"
declare -i i
i=0
for v in $@
do
	i=i+1
	echo "第$i個參數        :$v"
done

函數 

使用
function funcName(){
	...
}
定義 函數
使用 
funcName p1 p2 ... pn 調用函數
函數 讀取參數的方法 同 腳本讀取參數

example 

function Show(){

	declare -i i
	i=0
	for v in $@
	do
		i=i+1
		echo "第$i個參數        :$v"
	done
	
	return 1
	echo "不會顯示 已經返回"
}
