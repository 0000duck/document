基础正则表达式

特殊符号含义
[:alnum:]英文大小写 数字
[:alpha:]英文大小写
[:blank:]空格 [tab]
[:cntrl:]键盘控制键 [cr] [lf] [tab] [del]
[:digit:]数字
[:graph:]除 空格 [tab] 外 所有按键
[:lower:]小写英文
[:print:]任何可被打印出的字符
[:punct:]标点 ' ? , # $ ...
[:upper:]大写英文
[:space:]任何会产生空白的字符 空格 [tab] [cr]
[:xdigit:]十六进制数字


RE字符含义
^wordword在行首
word$word在行尾
.一定有一个任意字符
\转义字符
c*重复前一个字符 0 到 任意次
[list]包含list中的一个字符
[n1-n2]包含[n1,n2]连续编码的一个字符
[^list]不包含list中的字符
[^n1-n2]...
c\{n,m\}连续[n,m]个前一个字符
\{n\} n个
\{n,\} [n,+∞)个

grep '正则表达式' filename
-a //将binary文件以text文件方式查找
-c //计算找到次数
-i //忽略大小写
-n //输出行号
-v //反向查找 既没有指定字符在显示出来

-A //后接数字 代表把找到的行 其后 指定行数也显示出了
-B //后接数字 代表把找到的行 其前 指定行数也显示出了
--color=auto

example
ls -al | grep "^a.*"
grep "^a.*" ~/ //从目录下所有文件中查找
grep "^a.*" ~/test //从指定文件中查找

sed
对指定行数据 进行 删除 增加 替换

sed [-nefr] [动作]
-n //使用安静模式 默认stdin的所有数据都会显示到屏幕 -n 只有sed处理过的[n1,n2]数据才显示
-e //直接在命令行上进行sed动作
-f //直接将sed动作写在一个文件内 -f filename 对文件进行sed
-r //使用扩展到正则表达式 （默认基础）
-i //直接修改读取的文件内容 而不是由屏幕输出

[动作] //动作部分要用 ' 括起来

sed 可接多个 [动作] 此时 每个动作前都需要添加 -e
[n1][,n2]function //在[n1,n2]行进行function
[n1]function //只处理n1行
[n1][$]function //n1行到末尾

function 参数
a //后接字符串 在目前行下列新添一行 写入指定字符串
c //后接字符串 用指定字符串替换[n1,n2]
d //删除[n1,n2]
i //后接字符串 在目前行上列新添一行 写入指定字符串
p //打印[n1,n2]
s //正则表达式替换 s/旧字符串/新字符串/g旧字符串可用正则表达式指定

example
ls -l | sed -n '1,2p' //动作部分要用 ' 括起来

RE字符含义

?0个或一个前一个字符
|逻辑或

()+多个重复的组 类似 +

printf
将每行数据每个字段安指定方式显示 字段以[tab]或和空格 隔开
不能用于管道
\a //输出警告声
\b //退个键
\f //清屏
\n //换行
\r //Enter
\t //水平[tab]
\v //垂直[tab]
\xNN //NN为两位数字 列出十六进制 NN 代表的字符

%s %i...同c语言

example
printf '%10s %5i %8.2f\n' $(cat a.txt)
printf '%10s %5i %8.2f\n' $(cat a.txt | grep "this is a test" )

awk
将数据每行以指定分隔符(默认空格 [tab])分成不同字段 多每个字段进行处理
awk '条件1{动作1}条件2{动作2}...' filename
$0 //整行数据
$1 //字段1
$2 //字段2

NF //第几行拥有的字段总数
NR //正在处理的是 第几行 数据
FS //当前分隔符

example
cat /etc/passwd | awk '{FS=":"}$3<10{print $1 "\t" $3}'
//将管道中 字段3数据 小于10的 行 字段1 指定3 打印 那之前改变分隔符为:

文件比较

diff [-bBi] from-file to-file
以行为单位比较文件
-b //忽略空格的不同 (a b) 和 (a     b)相同
-B //忽略空白行区别
-i //忽略大小写
结果
4d3 //左边第4行被删除了 基准是右边第三行
6c5 //左边第6行被替换成右边第5行

cmp [-s] file1 file2
以字节为单位比较文件
-s //将所有不同列出 默认只列出第一处
