关机 

sync	//同步写入磁盘
reboot	//重启
halt	//硬件关机 不建议 
poweroff	//硬件关机 不建议

shutdown 时间 警告信息 

-t sec	//过多少秒后关机
-k	//不真的关机 指数发出警告消息
-r	//重启
-h	//关机
-n	//不经过init程序 直接以 shutdown 关键
-f	//关机并开机后 强制略过fsck磁盘检查
-F	//系统重启后 强制fsck磁盘检查
-c	//取消shutdown操作
时间	//关键时间 如 now [现在] 20：30 [二十点 三十分] 10 [10分钟后]

切换执行等级 

init 0	//切换到等级0
	0	//关键
	3	//纯命令行
	5	//图像界面
	6	//重启

忘记root密码 

启动系统读秒时按下任意键 进入启动引导程序选择
点e 进入编辑模式
选择 kernel /vm... 此行 点e 编辑此行
在 kernel /vm... 最后添加single 之后 [enter](单用户模式)
点 b 此此选项进入 单用户模式
passwd 修改root 密码

man 命令 //查询命令用法 

man的查询文件在 
/etc/man.config(一些版本 为 man.conf 或 manpath.conf)

类型代号 

PASSWD(1) User Commands PASSWD(1)

1	用户在shel中可操作的命令或可执行文件
2	系统内核调用的函数与工具等
3	常用的函数 函数库 (大部分为c函数库)
4	设备文件说明
5	配置文件 或某些格式文件
6	游戏
7	惯例与协议等
8	系统管理员可用的命令
9	跟kernel有关

说明代号 

NAME	//简短命令 数据名称说明
SYNOPSIS	//简短的命令执行语法简介
DESCRIPTION	//完整说明
OPTIONS	//针对SYNOPSIS列举所有可用选项说明
COMMANDS	//当此软件在执行时 可在此软件中执行的命令
FILES	//此程序或数据所使用或参考或连接到的某些文件
SEE ALSO	//相关的其他说明
EXAMPLE	//范例
BUGS	//是否有相关错误

info
