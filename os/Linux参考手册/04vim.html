vi工作模式
vi有三种工作模式
一般模式 用于光标的移动 数据的删除 粘贴
编辑模式 输入文本内容
命令行模式 读取 保存 查找 设置编辑环境

vi进入编辑模式

commandnote
i,Ii从目前光标位置插入
I从目前光标所在行第一个非空字符出插入
a,Aa从目前光标位置下一个位置插入
A目前光标所在行最后一个字符出插入
o,Oo光标下一行插入
O光标上一行插入
r,R进入替换模式
r仅替换一次
R一直替换

vi命令

commandnote
:w保存
:w!强制保存
:q退出
:q!...
:wq保存并退出
ZZ若文件未改动离开 若文件改动了保存后离开
:w [filename]另存为
:r [filename]将filename文件内容 插入到光标后面
:n1,n2 w [filename]将[n1,n2]行数据 保存为filename
:! command执行bash指令
:set nu显示行号
:set nonu隐藏行号

环境设置
vim中通过 :set ... 设置vim环境
在~/下建立.vimrc文件 可可设置vim环境


setnote
:set nu
:set nonu		是否显示行号
:set hlsearch
:set nohlsearch		搜索内容是否高光
:set autoindent
:set noautoindent		是否自动缩排
:set backup是否备份文件 默认noback
若备份 文件被改写时 会被备份为 file~
:set ruler是否显示状态栏
:set showmode是否显示--INSERT--等信息
:set backspace=(012)一些版本linux不允许在编辑模式下用backspace删除
此设置为2 可以删除
0或1 仅可删除刚才输入的字符
:set all显示全部可设置的参数
:set显示与默认值不同的设置
:syntax on
:syntax off		是否根据文档类型以不同颜色显示其内容
:set bg=dark
:set bg=light		可以显示不同颜色的色调

other
vim 会在编辑文件时为文件创建 .file.swp文件用于文档异常退出时 恢复
若要恢复 重新打开文件 在出现的选择选 (R)ecover
vim或为使用它的用户创建 ~/.viminfo 其中记录了其所有的vim操作

dos2UNIX [-kn] file [newfile] //将dos格式文件转化为unix文件
UNIX2dos [-kn] file [newfile] //...
k 不改变原文及的mtine
n 保留原文件 将转换后的文件保存到 newfile
