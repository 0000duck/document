virtual box 

virtual_box 一個 開源 跨平台 免費 的虛擬機軟件

可以虛擬 Solaris Windows DOS Linux OS/2 Warp OpenBSD FreeBSD android等os
可以支持 usb2.0接口 但需要安裝 Virtualbox Extension Pack

其在3D渲染上 無法和 VMWare相比 然其消耗資源小 特別適合 安裝存終端的 Linux

官網 https://www.virtualbox.org/

端口映射 

主機 填入 本機ip 而非 虛擬os的網關地址

android 

1	從 https://code.google.com/p/android-x86/downloads/list 
	下載最新的 x86 iso鏡像安裝文件
2	虛擬機新建 FreeBSD 系統

常用命令 

//為鏡像產生新的uuid
VBoxManage  internalcommands sethduuid "E:\virtualbox\sb\sb.vdi"
