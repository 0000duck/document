TextView 

TextView 是一個 文本顯示框 可顯示 單/多行 文本
同時 支持 在文本 上/下/左/右 顯示一個 指定的 圖片

常用方法 xml屬性

xml attrfunctionnote
android:autoLink
none 0x00
web 0x01
email 0x02
phone 0x04
map 0x08
all 0x0f		setAutoLinkMask(int)將文本轉化為 可單擊的 url 鏈接
android:drawableBottomsetCompoundDrawablesWithIntrinsicBounds(int,int,int,int)在文本下方繪製指定 圖像
android:drawableTopsetCompoundDrawablesWithIntrinsicBounds(int,int,int,int)在文本上方繪製指定 圖像
android:drawableLeftsetCompoundDrawablesWithIntrinsicBounds(int,int,int,int)在文本左方繪製指定 圖像
android:drawableRightsetCompoundDrawablesWithIntrinsicBounds(int,int,int,int)在文本右方繪製指定 圖像
android:singleLine
true
false		setTransformationMethod(TransformationMethod)是否只顯示單行
android:inputType
setRawInputType(int)文本顯示 方式 如 textPassword... 多個值用 | 分隔
android:hint
setHint(int)文本為空時 顯示的 內容
...

ImageView 

ImageView 用來 顯示任意一個 Drawable 對象 通常是一副 圖片

常用方法 xml屬性

xml attrfunctionnote
android:srcsetImageResource(int)設置 要繪製的 對象
android:adjustViewBoundssetAdjustViewBounds(boolean)是否要調整 ImageView 的邊界以 保持要顯示對象的 長寬比
android:maxHeightsetMaxHeight(int)設置ImageView 最大高度(必須adjustViewBounds 為true 才生效)
android:maxWidthsetMaxWidth(int)設置ImageView 最大寬度(必須adjustViewBounds 為true 才生效)
android:scaleType
matrix 0
fitXY 1(縮放圖片 使之完全適應 ImageView)
fitStart 2(保存 比例縮放圖片 使之完全顯示到 ImageView中 之後 放在ImageView左上角)
fitCenter 3(... 中間)
fitEnd 4(.. 右下角)
center 5(不縮放 將圖片放到 ImageView中間)
centerCrop 6(保存比例縮放 使圖片能完全覆蓋 ImageView)
centerInside 7(... 能夠完全顯示到 ImageView)		setScaleType(ImageView.ScaleType)設置如何 縮放 對象
...

public class MainActivity extends Activity {
	@Override
	protected void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.activity_main);
		
		ArrayAdapter<CharSequence> adapter = ArrayAdapter.createFromResource(this, R.array.wtype, android.R.layout.simple_list_item_1);
			/*simple_list_item_1	每個列表項 一個 文本
			simple_list_item_2	同simple_list_item_1 不過 字體略大
			simple_list_item_checked	... 已夠選定列表項
			simple_list_item_multiple_choice	... 帶多選框 文本
			simple_list_item_choice	... 帶單選框 文本
			*/
		//設置 適配器
		ListView listview = (ListView)findViewById(R.id.list_view);
		listview.setAdapter(adapter);
		
		listview.setOnItemClickListener(new OnItemClickListener(){

			@Override
			public void onItemClick(AdapterView<?> parent, View view, int position, long id) {
				//獲取 單擊項
				String result = parent.getItemAtPosition(position).toString();
				Toast.makeText(MainActivity.this,result, Toast.LENGTH_SHORT).show();
			}
			
		});
	}
