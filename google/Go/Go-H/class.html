嵌入 聚合 

Go 不支持 繼承 重載 Go只支持嵌入 聚合

example 

package main

import (
	"fmt"
)

type Point struct {
	X, Y int
}

func (p *Point) Where() {
	fmt.Println(p.X, p.Y)
}

type cat struct {
	//嵌入
	Point

	//聚合
	name string
}

func main() {
	c := cat{Point{1, 2}, "tom"}
	fmt.Println(c)

	//对于嵌入可直接使用 其内部定义
	//fmt.Println(c.Point.X,c.point.Y)
	fmt.Println(c.X, c.Y)
	c.Where()
}

interface 

interface 接口 是一些方法 定義聚合
interface 同樣支持 嵌入

type interfaceName interface{
	funcName
	...
}
