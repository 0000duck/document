Revel 環境配置 常用命令

Revel 

Revel 一個高生產力的 開源(MIT) golang web框架

官網http://revel.github.io/index.html

環境配置 

1	下載revel框架 go get github.com/revel/revel
2	下載 revel 工具 go get github.com/revel/cmd/revel
3	到 github.com/revel/cmd/revel 目錄下 執行 go build 編譯 revel 工具
4	將 revel 工具 加入 環境變量 PATH
5	創建應用 revel new myapp

注意 

在 go get github.com/revel/cmd/ 時 如果報錯 
(package golang.org/x/net/websocket: unrecognized import path "golang.org/x/net/websocket")
可以 下載 https://github.com/golang/net.git 將 其中的 websocket
拷貝到 golang.org/x/net/ 下

常用命令 

//創建新項目
revel new 項目名

//運行 項目
revel run 項目名

//打包 revel 應用
revel package 項目名

//清除項目 臨時文件
revel clean 項目名

//查看 命令 幫助
revel help
