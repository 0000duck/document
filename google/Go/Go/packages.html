packages 包 

每個 Go 程序都由 包組成
程序入口點是 main

使用 import 引入包
(可像示例中impot 多個包 亦可分別 import
import "fmt"
import "math/rand"
import bar "bytes"	//導入包 並為使用 bar覆蓋其默認包名
)

example 

package main

import (
	"fmt"
	"math/rand"
)

func main() {
	fmt.Println("My favorite number is", rand.Intn(10))
}

注意 

包中被導出的名稱 首字母必須大寫(大寫既自動被導出) 如 
math.Pi (正確寫法)
math.pi	(錯誤寫法)

function 函数 

//Go 使用func 定义函数 返回值型别 在函数定义之后
func add(x , y int) int {	//同型别参数 只需要在最后写出型别
	return x + y
}

Go 函数 可以返回多个值 

func main() {
	l, r := swap("left", "right")
	fmt.Println(l,r)
}

func swap(left, right string) (string, string) {
	return right, left
}

可以给返回值 命名 

//一个空 return 将返回 被命名的 返回值
func main() {
	l, r := swap("left", "right")
	fmt.Println(l)
	fmt.Println(r)
}

func swap(left, right string) (l, r string) {
	l = right
	r = left
	return
}

var 变量 

Go 使用 var 定义变量
var x,y,z int
var x,y,z int = 1,2,3

x,y := 1,2	相当于c++ auto

內建函數 

Go 預定義了少量函數 無需引入任何包 就可以使用
close	用於關閉channel
delete	刪除map key
len 	返回slice 字符串長度
cap		返回slice 數組 函數 詳細信息
new		分配內存
make	創建內建型別 map slice channel ...
copy	複製slice
append	追加slice
panic recover	用於異常機制
print println	底層打印函數 主要用於調試
complex real imag	用於複數處理
