code-prettify 使用

code-prettify 

code-prettify 是google 提供的一個 用於 在 html上 高光顯示 代碼的 開源(Apache License 2.0) js 庫 

官網 https://github.com/google/code-prettify

使用 

1	將 code-prettify-master/src 加入項目
	<link rel="stylesheet" type="text/css" href="../src/prettify.css">
	<script src="../src/prettify.js"></script>

2	將可選的 顯示風格 引入 
	code-prettify-master/styles/ 下提供了 4個 默認 風格
	desert.css
	doxy.css
	sons-of-obsidian.css
	sunburst.css

3	在html中定義一個 pre 或 code 標籤 並指定 class='prettyprint'
	如果需要行號 class='prettyprint linenums'
	指定行號 起始索引 class='prettyprint linenums:5'

4	在html加載完成後調用 prettyPrint 函數
