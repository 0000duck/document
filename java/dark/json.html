JSonObject 

import dark.data.format.JSonObject;

方便生成 json 字符串 以便於 將信息返回給js

function 

//返回 "text/javascript;charset=UTF-8"
public static String getJSonContext()

//設置一個 json 數據隊(name 為隊名 value 為值 可以說java基本型別 字符串 java.util.HashMap)
public void put(String name,Object value)

//返回一個 json 名字 對應 值
public Object get(String name)

//刪除一個 json 值隊
public void erase(String name)

//返回內部以馬匹 保存的 json 數據
HashMap<String,Object> get()

//將 json 數據 專為 字符串 形式
public String toString()

注意 

JSonObject 內部 以HashMap 存儲數據
HashMap 未保證線程安全 故多線 需要 自己同步

example 

System.out.println(JSonObject.getJSonContext());
        
        //实例化
        JSonObject json = new JSonObject();
        
        //添加一般属性
        json.put("name_1", "name_1");
        json.put("name_2", "name_2");
        json.put("name_1", "name_3");   //重新设置值
        
        //删除属性
        json.put("name_3", "name_3");
        json.erase("name_3");
        
        //json.clear();
        
        //添加 json {} 属性
        HashMap h0 = new HashMap();
        h0.put("id", "0");
        h0.put("data", "d0");
        json.put("json", h0);
        
        //添加数组 属性
        ArrayList list = new ArrayList();
        HashMap h1 = new HashMap();
        h1.put("id", "1");
        h1.put("data", "d1");
        
        list.add(h0);
        list.add(h1);
        json.put("data", list);
       
        
        //获取 json 字符串
        System.out.println(json);
