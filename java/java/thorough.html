繼承 

java 只能單繼承(弱爆)
但java有 所謂的接口 子類可以繼承多個接口

//定義一個child類 派生自 father
public class child extends father
{};

java 使用 super. 訪問被子類覆蓋的數據

java使用 abstract 定義抽象類 和抽象方法(簽名沒有 優雅的 =0)

在類名前 加上 final 的類 不能被其他類繼承

interface 

java中 所謂的接口 既只有 static final 變量 靜態變量 和 方法簽名

public interface interface1
{
static final String name="name";
void get_name();
}

//繼承接口使用 implements
class child implements interface1
{}
class child extends father implements interface1,interface2,interface3...
{}
