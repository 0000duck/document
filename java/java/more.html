sort

    public class test implements  Comparable
    {
        int x;

        @Override
        public int compareTo(Object t) {
            //返回負 <
            //...0 ==
            //...正 >
            test tmp  =   (test)t;
            return this.x - tmp.x;
        }
        
    }

字符串 

java 使用class String 表示字符串
String 的操作 只是創建一個新的引用到字符串 而不改變自身內存

StringBuilder 操作基本同 String 只是 其改變字符串時 是為自己分配內存 之後拷貝字符串到內存
StringBuffer 基本同StringBuilder 不過其操作都是 線程安全的

StringTokenizer 

import java.util.StringTokenizer;

class StringTokenizer 是java中 用於分解字符串的類

String[] String::split(String regex) 亦可分解字符串

example 

String str   =   "1911-10-10";
StringTokenizer  token   =   new  StringTokenizer(str,"-");

while(token.hasMoreTokens())
{
   System.out.println(token.nextToken());
}

java 亦提供了對正則表達式的支持

//matches 返回字符串 是否和 傳入的正則表達式 匹配
boolean String::matches(String regex)

Pattern Matcher 

java 以提供了專門處理正則表達式的 Pattern Matcher
使用 Pattern 代表一個正則表達式 Matcher記錄匹配結果

example 

String str   =   "1911-10-10";
Pattern regex    =   Pattern.compile("1.*0");
Matcher matcher = regex.matcher(str);

if(matcher.matches())
{
    System.out.println("matches");
}
