java 安裝java 常用java命令

java 

java 是一個跨平台 完全面向對象的 解釋執行語言
官網http://www.java.com/

java 主要包含三個版本
SE	標準版 包含核心實現
EE	企業版 包含SE全部內容 以及網絡服務等組件
ME	對手機 PDA設備 TV機頂盒 以及嵌入設備等

安裝install 

若只需要運行java 從官網下載jre並安裝

若需要開發 從官網下載 jdk 並安裝 並設置java執行的bin路徑
http://java.sun.com

之後神人就可以使用 vim 或 記事本進行 編程
否則可以選擇安裝個 eclipse 或 NetBeans 或其他IDE

常用java命令 

java -version	//查看版本
javac *.java	//編譯 java 字節碼
java path	//執行 java 桌面程序
