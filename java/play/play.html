play 環境配置 常用命令

play 

play 是一個輕量級的 開源(Apache 2 License) java/scala web framework

官網https://www.playframework.com/

環境配置 

1 安裝 jdk 1.6或以後的 java環境

2 安裝sbt(play 使用sbt編輯 java代碼) http://www.scala-sbt.org/download.html

3 從play官網下載 最新支持 腳本 typesafe-activator-1.2.12-minimal.zip 解壓 並且加入 PATH 環境

4 執行 activator -help (自動下載其他需要的 包 如果已經下載 顯示 幫助信息)


3 4 可簡化為 下載 Offline Distribution(包含所有 包依賴) 版本 並解壓 
在解壓目錄 運行 activator -help
將目錄 加入 path 環境

常用命令 

//創建一個java項目
activator new my-first-app play-java

//創建項目 使用 play ui
activator ui

//運行 web
activator run
//在80端口運行
activator "run 80"

//以debug方式 運行 web
activator test

//為 項目 創建 play ui
activator ui

注意 

創建項目時 或從 play官網下載模板
在 優越的社會主義bitch國家 是必須設置代理才能訪問的

編輯
C:\Users\Administrator\.activator\activatorconfig.txt
或者
~/.activator/activatorconfig.txt
增加如下代理設置

# This are the proxy settings we use for activator
-Dhttp.proxyHost=PUT YOUR PROXY HOST HERE
-Dhttp.proxyPort=PUT YOUR PROXY PORT HERE
# Here we configure the hosts which should not go through the proxy.  You should include your private network, if applicable.
-Dhttp.nonProxyHosts="localhost|127.0.0.1"
# These are commented out, but if you need to use authentication for your proxy, please fill these out.
#-Dhttp.proxyUser=PUT YOUR PROXY USER HERE
#-Dhttp.proxyPassword=PUT YOUR PROXY PASSWORD HERE
