bootstrap 

bootstrap 最初由Twitter的 Mark Otto 和 Jacob Thornton 編寫
之後 Twitter 開發者 將其開源(MIT)出來的 html 前端 樣式庫
bootstrap 支持響應式佈局 可以自動 適應 pc 手機 平板

官網 http://getbootstrap.com/

bootswatch 

bootstrap 只提供了一個 界面配色風格 而且是 明亮系的
對於優秀的 人類來說 暗黑系才是 最舒服的

bootswatch 是從bootstrap 建立的 分支 爲bootstrap 提供了 各種 配色風格
用 bootswatch 提供的 bootstrap.min.css 替換掉 bootstrap的 原本 css 即可 爲 bootstrap 使用不同的風格

官網 http://bootswatch.com/
