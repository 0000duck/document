pac 是一個 javascript 文件
其中 定義了 函數 function FindProxyForURL(url, host) 
猶 FindProxyForURL 的返回值 決定 如何 訪問 網絡

FindProxyForURL 返回值 

//直接訪問 網絡
return "DIRECT"; 

//使用 http 代理
return "PROXY 192.168.1.1:3128";
return "PROXY mydemon.com:3128";

//使用 socket5 代理
return "SOCKS5 lilinux.net:1080";

//使用 多個代理(如果前面的代碼無法使用 則連接 後續代理)
return "DIRECT; PROXY 192.168.1.1:3128; SOCKS5 lilinux.net:1080";
