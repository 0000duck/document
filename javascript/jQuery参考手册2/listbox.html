jQuery-Multi-Select-List-Box-Plugin-listbox-js 

jQuery-Multi-Select-List-Box-Plugin-listbox-js 是一個 listbox 插件 
實現了 基本的 listbox(支持多選 單選) 功能 並可以添加一個 搜索框 搜索列表項

官網 http://www.jqueryscript.net/form/jQuery-Multi-Select-List-Box-Plugin-listbox-js.html

獲取支持 

1	從官網下載最新版本 jQuery-Multi-Select-List-Box-Plugin-listbox-js.zip 並解壓
2	將src\listbox.css listbox.js 引入項目
3	在html中編寫 select 標籤
4	$('select').listbox()	將select替換為listbox

scripts
$(document).ready(function(){
	
	$('#listbox').listbox(
		//下面參數皆為可選參數
		{'searchbar': true,			//是否添加搜索框
		'class': "dark_listbox"		//附加的class屬性
		}	
	);
	
	$("#cmd").keydown(function(e) {
        if(e.keyCode == 13){
			var cmd	=	$(this).val();
			cmd	=	$.trim(cmd);
			if(cmd != ""){
				command(cmd)
			}
		}
    });
	
	//當選項發生改變
	$('#listbox').change(function(e) {
        alert($(this).val());
    });
	
	
});
function command(cmd){
	if(cmd	==	"show"){
		alert($("#listbox").val());		//返回選中項
	}
	else if(cmd == "select"){
		var element	=	$('#listbox').next(".lbjs:first");
		//清除 外在 效果
		element.find(".lbjs-item[selected='selected']").removeAttr("selected");
		//清除 選項值
		$('#listbox').val(null);		
	}	
}

注意 

listbox() 方法並為替換原select 而只是將其隱藏 
然後在 select 之後 添加一個 div 並在此div中 顯示listbox

dark_listbox_wrap 

jQuery-Multi-Select-List-Box-Plugin-listbox-js 未提供完備的 js操作列表項 的能力 
dark_listbox_wrap 對齊進行了包裹 以提供這些能力

1	引入 jQuery-Multi-Select-List-Box-Plugin-listbox-js 需要環境
2	引入 dark_listbox_wrap.js
3	使用 jQuery-Multi-Select-List-Box-Plugin-listbox-js 構建listbox
4	調用 dark_listbox_wrap.create($("select"))	//傳入 被 jQuery-Multi-Select-List-Box-Plugin-listbox-js 替換的 listbox jQuery元素 (記錄其引用)
5	調用 包裹方法
