HtmlBox 使用 toolbars 自定義工具 皮膚 皮膚文件夾位置 常用方法

HtmlBox 

HtmlBox 一个基于 jQuery的 在线 富文本编辑器
可以將 textarea 專為 文本編輯器

官網http://remiya.com/htmlbox/

使用 

1	從官網下載 最新代碼 HtmlBox_4.0.3.zip 
2	解壓 將 image文件夾(圖像資源) 複製到項目目錄
3	將 htmlbox.min.js 導入html
4	創建一個 textarea
5	使用 htmlbox 方法 替換textarea 為富文本框(htmlbox 傳入json 指定各種 創建屬性)

example 

<script language="javascript" src="scripts/HtmlBox_4.0.3/jquery-1.3.2.min.js"<>/script>
<script language="javascript" src="scripts/HtmlBox_4.0.3/htmlbox.min.js"<>/script>
<script language="javascript" >
$(document).ready(function(){
	
	$('#mttextbox').htmlbox({
		toolbars:[
			["bold","italic","underline"]
		]
		,about:false	//去掉幫助?按鈕
		}
	);
});
</script>

注意 

使用的 jQuery版本 必須為 HtmlBox_4.0.3.zip 壓縮包中的版本

toolbars 

toolbars 為工具欄定義 
以字符串名字 定義工具欄中的 工具 ["separator","cut","copy"]
多個工具欄使用 , 隔開

separator	分隔符
underline	下劃線
undo		撤銷
redo		逆撤銷
justify		取消設置的對齊
...
詳見源碼中 demo_full.html

example 

$(document).ready(function(){
	$('#mttextbox').htmlbox({
		toolbars:[
			["separator","cut","copy","paste","separator","bold","italic","underline"]
			,["separator","undo","redo","separator","left","right","center","justify"]
		]
		}
	);
});

自定義工具 

HtmlBox 支持自定義 工具欄 需要用一個json數據定義
顯示 圖標		icon
顯示 文本		tooltip
單擊 回調函數		command

example 

$(document).ready(function(){
	var save_tools	=	{
		icon:"save.gif"
		,tooltip:"save"
		,command:function(){alert("saving");}
		}
	$('#mttextbox').htmlbox({
		toolbars:[
			[save_tools,"separator","cut","copy","paste","separator","bold","italic","underline"]
			,["separator","undo","redo","separator","left","right","center","justify"]
		]
		}
	);
});

注意 

圖標路徑 不是以當前文件為相對路徑 而是 HtmlBox使用的 皮膚文件夾(images\default\)為相對路徑

皮膚 

HtmlBox支持 sliver(default) blue green red 4中皮膚

example 

$(document).ready(function(){
	var save_tools	=	{
		icon:"save.gif"
		,tooltip:"save"
		,command:function(){alert("saving");}
		}
	$('#mttextbox').htmlbox({
		toolbars:[
			[save_tools,"separator","cut","copy","paste","separator","bold","italic","underline"]
			,["separator","undo","redo","separator","left","right","center","justify"]
		]
		,skin:"red"
		}
	);
});

皮膚文件夾位置 

使用 icons 屬性 可以改變 皮膚文件夾位置
(只能是 images[使用的當前目錄]的1級子目錄)

使用 idir 設置 使用的 當前目錄

example 

$(document).ready(function(){
	var save_tools	=	{
		icon:"save.gif"
		,tooltip:"save"
		,command:function(){alert("saving");}
		}
	$('#mttextbox').htmlbox({
		toolbars:[
			[save_tools,"separator","cut","copy","paste","separator","bold","italic","underline"]
			,["separator","undo","redo","separator","left","right","center","justify"]
		]
		,skin:"red"
		,icons:"silk"
		,idir:"image2"
		}
	);
});

//js
$(document).ready(function(){
	
	var edit1	=	$('#mttextbox1').htmlbox();
	var	edit2	=	$('#mttextbox2').htmlbox();
	
	$('input').click(function(e) {
		var html	=	edit1.get_html();
		edit2.set_text(html);
		alert(html);
	});
});
