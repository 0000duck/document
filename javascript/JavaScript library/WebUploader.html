WebUploader 環境配置 example

WebUploader 

WebUploader 是一個 功能齊全 開源(BSD) 的 javascript 上傳組件
由 中華淪陷區 的百度公司的fex小組 開發

WebUploader 沒提供任何 ui 全部功能以 api的方式 提供
WebUploader 默認使用 html5 同時在不支持html5時 使用 flash進行上傳

官網 http://fex.baidu.com/webuploader/
github https://github.com/fex-team/webuploader

環境配置 

1	從官網下載最新源碼 並解壓
2	將 webuploader.nolog.min.js webuploader.css 引入項目

注意 

webuploader.js 會將一些日誌 發送到 百度 nolog版本 則不會 發送

example 

//將 js css 引入項目
<link rel="stylesheet" type="text/css" href="/public/js/webuploader/webuploader.css">
<script type="text/javascript" src="/public/js/webuploader/webuploader.nolog.min.js"></script>


//在html中 創建一些 必要的 標籤 作爲 webuploader 容器
<div id="uploader" class="wu-example">
	<!--存放檔案信息的容器-->
	<div id="thelist" class="uploader-list"></div>
	<div class="btns">
	<div id="picker">選擇檔案</div>
	<button id="ctlBtn" class="btn btn-default">開始上傳</button>
</div>
