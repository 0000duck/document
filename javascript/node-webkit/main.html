node-webkit 環境配置 Hellow World package.json

node-webkit 

node-webkit
node-webkit 一個將 webkit(現在使用 Chromium) node 結合
以 使用 html js 開發 桌面 程序的 開源(MIT) 跨平台 框架

官網 https://github.com/nwjs/nw.js/

環境配置 

1	從官網下載 最新 release 包 並解壓
2	將 解壓的包 加入 PATH 變量

Hellow World 

1	編輯 一個 html 作為 入口

<!DOCTYPE html>
<html>
<head>
	<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
	<title>Hellow World</title>
</head>
<body>
Hellow World
</body>
</html>



2	編輯 package.json 配置 webkit 容器
{
  "name": "nw-demo",
  "version": "0.0.1",
  "main": "index.html"
}



3	執行 nv package.json/. 運行 程序
	亦可 將 代碼 打包到 zip中 運行

package.json 

{

	"main": "index.html",	//入口點

	"name": "nw-demo",		//全局 唯一 應用名 小寫字符串

	"description": "demo app of node-webkit",
	
	"nodejs":true,		//是否啟用 node.js
	"single-instance":true,	//是否只允許1個實例

	"version": "0.1.0",

	"keywords": [ "demo", "node-webkit" ],

	"window": {

		"title": "node-webkit demo",	//窗口標題

		"icon": "link.png",

		"toolbar": true,	//是否創建 導航欄

		"frame": false,

		"width": 800,

		"height": 500,

		"position": "mouse",	//窗口打開 位置 null center mouse

		"min_width": 400,

		"min_height": 200,

		"max_width": 800,

		"max_height": 600,

		"resizable":true,	//是否可調整窗口大小 

		"always-on-top":false,	//是否始終前置

		"fullscreen":false,	//全屏

		"show_in_taskbar":true	//在系統任務欄上顯示 圖標

	},

	"webkit": {
		"plugin": true
	}

}
