函数的定义 函数的参数 函数的返回值 构造函数 常用函数

函数的定义 

方式一
function sum(a,b){
	return a+b;
}
var sum = function(a,b){
	return a+b;
}

方式二
//var name=new Function([参数,...],函数体);
var sum = new Function("a","b","return a+b;");
sum(1,2);

注意 

函数的参数可以是另外的函数 
可用 if(参数 instanceof Function) 判断传入的是否是函数

javascript中的函数都是对象Function的实例

函数的参数 

arguments对象代表了当前被调用的函数的参数数组 其length代表了实际传入参数数
fun.length代表了函数定义的参数数

參數 是基本型別則為 值傳遞 複合型別為應引用傳遞

函数的返回值 

对于基本型别javascript函数返回值的副本

对于复合型别javascript函数返回 引用
javascript还可以返回函数的引用(函数 是复合型别 Function的实例)

构造函数 

对于javascript没有类的概念 只需定义出构造函数就可将其看成一个'对象类'

example 

function A(name){
	this.name=name;
	this.getName=getName;
}
function getName(){
	return this.name;
}

var a=new A("123");
alert(a.getName());
