screen对象 

通过screen对象获取屏幕信息 从而可写出兼容 不同分辨率的程序

屏幕分辨率高度

	
属性和方法描述

	
availWidth屏幕有效宽度

	
availHeight屏幕有效高度

	
width屏幕分辨率宽度

	
height

	
colorDepth显示器颜色位

	
javaEnabled()返回是否支持java (用于JavaApplet使用前检测)

注意 

有效高度宽度指 分辨率减去系统必要占用高度宽度
(如windows系统中要减去开始栏)
动态设置网页布局 應用有效高度宽度
