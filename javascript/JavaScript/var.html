Date 

Date 是JavaScript中对时间日期的包装



//以当前时间日期 构造Date
date=new Date();

//以相对1970-01-01 0:0:0的相对毫秒数 构造Date
date=new Date(val);

//以指定时间日期构造
date=new Date(year,month,date[,hour[,min[,sec[,ms]]]]);
