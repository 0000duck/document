	}

字符结构

字符结构描述
.任意字符
X单个字符(包括控制字符\t \n \r \\ \f)
Xxx字符串
[abc]包含字符a或b或c 的单个字符
[^abc]不包含字符a或b或c 的单个字符
[a-zA-Z]a到z 或A到Z 的单个字符
\s空白符(空格 跳格 回车 换行 换页)
\S非空白符
\d数字(0~9)
\D非数字
\w文本字符[a-zA-Z0-9_]
\W非文本字符

定义

定义描述
^X以字符X开始
X$以字符X结尾
XY字符Y紧跟在字符X后
X|Y字符X或Y
X?空字符串或最多一个X
X*空字符串或一个到多个X
X+一个或多个X
X{n}n个X
X{n,}至少n个X
X{n,m}n到m个X
