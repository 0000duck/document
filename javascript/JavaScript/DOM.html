DOM HTML DOM

DOM 

DOM接口可以操作所有符合DOM规范的文档

访问节点 

//返回指定id的节点
document.getElementById(id);

//返回指定name的节点数组
document.getElementsByName(name);

//返回指定标签的节点数组(tag为*返回全部节点)
document.getElementsByTagName(tag); 



//第一个子节点
document.firstChild;

//最后一个子节点
document.lastChild;

//返回是否有子节点
document.hasChildNodes();

//子节点数组
document.childNodes;

处理节点

方法或属性描述
node.nodeName节点名(如 DIV)
node.nodeType节点类型(如 1)
node.setAttribute(name,value);
node.getAttribute(name);

修改创建节点 

//创建文本节点
document.createTextNode(data);

//根据指定标签名创建节点
document.createElement(tagName);

//创建文档碎片
document.createDocumentFragment();



//删除指定子节点
node.removeChild(node);

//在指定子节点前添加节点
node.insertBefore(insertNode,whereNode);

//用指定节点替换子节点
node.replaceChild(inserteNode,replaceNode);

//添加一个节点到子节点尾
node.appendChild(app);
