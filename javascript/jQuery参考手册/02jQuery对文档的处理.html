移动 添加 替换 包裹 删除

移动 添加

方法描述
$(selecotr1).append($(selecotr2))把s2移动到s1 成为其子元素末尾
$(selecotr1).appendTo($(selecotr2))把s1移动到s2 成为其子元素末尾
$(selecotr1).prepend($(selecotr2))把s2移动到s1 成为其第一个子元素
$(selecotr1).prependTo($(selecotr2))把s1移动到s2 成为其第一个子元素
$(selecotr1).append($(selecotr2))
...
$(selector1).after($(selector2))把s2移动为s1兄弟元素 在s1后面
$(selector1).inserAfter($(selector2))把s1移动为s2兄弟元素 在s2后面
$(selector1).before($(selector2))把s2移动为s1兄弟元素 在s1前面
$(selector1).insertBefore($(selector2))把s1移动为s2兄弟元素 在s2前面

注意 

可用 $(html) 生成jQuery对象 如$(<p />) 生成了一个 p标签
$(selector1)为数组时 其每个元素都会被加上 selector2
移动后对象就不在原位置了 $(selector1)
想要保存selector2在原位 用 $(selector).clone()返回selector2的副本 
如$(selector).after($(selector).clone());

替换 包裹 删除

方法描述
$(selector1).replaceWith($(selector2))把selector1全删除 把selector2添加到最后一个selector1的位置
$(selector1).replaceAll($(selector2))用selector1替换全部的selector2
...
$(selector1).wrap($(selector))为selector1添加父元素(selector2的第一个匹配元素)
$(selector).unwrap()去掉selector的父元素(但其父元素的文本依然被保留)
$(selector1).wrapAll(selector2)把selector1全部移动到一起 再用selector2的第一个元素成为其父元素
$(selector1).wrapInner(selector2)为selector1的文本和每个后代添加父元素selector2(selector的第一个元素)
...
.remove()从html中删除本元素 但会返回本元素以供用于其他 如添加到html其他位置(appendTo...)
.detach()同.remove（）
.empty()删除元素的所有后代 只留下元素本身
