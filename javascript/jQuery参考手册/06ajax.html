ajax ajax全局设置 other

ajax 

$.ajax({
	url:"请求的URL地址",
	dataType:"html",	//服务器传回数据类型 可以是 html xml json script
	data:{"name":"test","id":"123"},	//向服务器传输的数据
	type:"get",	//请求的方式post get 默认get
	success:function(data){}	//请求成功时代处理函数
})

example 

$.ajax({
	url:"example.jsp",
	dataType:"html",
	type:"post",
	data:{"name":"test"},
	sucess:function(data){
		alert(data);
	}
});

other 

若$.ajax的dataType属性是script 可以接受一段javascript
这段代码将作为全局javascript
比如接受的js中定义了一个变量 于是可以sucess函数中直接使用这个变量

ajax快捷方法 

jQuery封装了三个快捷的方法 以让我们可以填入更少的参数
$.get(url,[,data][,sucess],[dateType]);	//其实就是默认get传输
$.post(url,[,data][,sucess],[dateType]);	//...post...
$.get("get.test",{"id":"123"},function(){},"html");
