jQuery和DOM转换 

实现jQuery元素和标准DOM元素之间相互转换

DOM to jQuery 

以 $() 传入DOM元素 便会返回jQuery元素
$(document.getElementsByTagName("div"))

jQuery to DOM 

调用$().get()方法;



//返回所有元素为dom元素数组
var dom=$('li').get();

//返回所有元素中第一个为dom元素
var dom=$('li').get(0);

//返回所有元素中第二个为dom元素
var dom=$('li').get(1);

//返回所有元素中倒数第一个为dom元素
var dom=$('li').get(-1);

//返回所有元素中倒数第二个为dom元素
var dom=$('li').get(-2);

基本选择器 

jQuery中最常用的选择器

基本选择器

selectornoteexample
*选取所有元素$('*') 包括了html和body ...
tag选择指定标签名的所有元素$('div')
#id选择指定id的元素$('#12')
.class选择拥有指定class属性的所有元素
此selector效率低下 一般不单独使用 多和其他选择器组合使用		$('.mybox')

注意 

多个选择器同时使用 选择符合所有选择器要求的元素
$('div.mydiv') //选择了所有包含class为mydiv的div元素

多个选择器以,(逗号)隔开 选择符合其中之一的元素 ,相当于逻辑或
$('p,.mydiv')//选择了所有p标签和所有class属性为mydiv的元素

过滤选择器 

用在基本选择器之后 用于从selector选择后端内容中过滤不需要元素
若不指明基本选择器 默认基本选择器为 *

基本 过滤选择器

selectornoteexample
:first选取第一个元素$('p:first')
:last选取最后一个元素$('p:last')
:even选取索引值是偶数的元素$('p:even')
:odd选取索引值是奇数的元素$('p:odd')
:eq(index)选取索引值为index的元素$('p:eq(0)')
:gt(index)选取索引值大于index的元素$('p:gt(1)')
:lt(index)选择索引值小于index的元素$('p:lt(1)')
:not(selector)选取不与selector匹配的元素$('p:not(.myp)')
//选取了所有class属性不为myp的p标签
:header选取了所有标题元素$(':header') //如 h1 h2...
:animatedx选取所有正在执行的动画元素

子元素 过滤选择器

selectornoteexample
:first-child第一个子元素
:last-child最后一个子元素
:only-child选取唯一的子元素
:nth-child(odd)子元素索引为偶
:nth-child(even)子元素索引为奇
:nth-child(index)子元素索引为index
:nth-child(equation)子元素索引符合equation（表达式）$('p:nth-child(3n+1)')
//equation可自定义 n取值为0 1 2 3 ...

内容 过滤选择器

selectornoteexample
:contains(text)选取含义文本text的元素$("div:contains('hello')")
:has(selector)选取含有后代元素为selector的元素
:parent选取含有后代元素或文本的非空元素
:empty选取不包含后代元素和文本的空元素

可见性 过滤选择器

selectornoteexample
:hidden选取不可见元素
:visible选取可见元素

属性 过滤选择器

selectornoteexample
[attr]选取使用了attr属性的元素$('div[class]')
[attr=value]选取attr属性为value的元素
[attr!=value]选取attr属性不为value的元素
[attr~=value]选取attr属性用空格分隔的值中有一个为value的元素
[attr^=value]选取attr属性以value开始的元素
[attr$=value]选取attr属性以value结束的元素
[attr*=value]选取attr属性含有value的元素

层次选择器 由两个selector组合而成

selectornote
selector1 selector2从selector1的后代里选取selector2
selector1>selector2从selector1的子元素里选取selector2
selector1+selector2从selector1后第一个兄弟元素里选取selector2
selector1~selector2从selector1后的所有兄弟元素里选取selector2

表单选择器 用于方便的选取表达元素

selectornote
:input选取input textarea select button
:text选取[type=text]的input
:password选取[type=password]的input
:radio选取[type=radio]的input
:checkbox选取[type=checkbox]的input
:image选取[type=image]的input
:file选取[type=file]的input
:button选取[type=button]的input 和 button
:submit选取[type=submit]的input
:reset选取[type=reset]的input
/*我是分界线*/
:enabled选取可用的表单元素
:disabled选取不可用的表单元素
:checked选取被选中的input
一些浏览器会把 $(':checked') 误选为option
所以推荐用$('input:checked')
:selected选取被选中的option
