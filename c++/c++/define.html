宏 ## 連接 字符串 __VA_ARGS__ 可變長參數

宏 

c++ 從c 繼承來了 宏 使用 方式 基本 同 c

## 連接 字符串 

使用 ## 可以 將宏 和字符串連接

#define TEST_SHOW_1 puts("1");
#define TEST_SHOW_2 puts("2");
#define TEST_SHOW(XXX) TEST_SHOW_##XXX
int main()
{
    TEST_SHOW(1)    //show 1
    TEST_SHOW(2)    //show 2

    return 0;
}

__VA_ARGS__ 可變長參數 

c99 標準開始 c的宏支持了可變參數 
使用 ... 定義 可變長
使用 __VA_ARGS__ 使用 可變長

#define PRINTF(...) printf(__VA_ARGS__);
int main()
{
    PRINTF("ok %d",1);

    return 0;
}
