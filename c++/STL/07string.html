string 

#include <string>

凡是以 一个索引和一个长度作为参数 的地方 字符串遵循以下两规则
1.	索引值必须合法
	对于搜索函数 可接受超过实际字符数的索引 这是会返回 string::npos
2.	字符长度可任意 若大于实际剩余字符数 或用 string::npos 表示剩余所有字符

string s(str,n)			//以str内开始于n的部分 当作s初值
string s(str,n,len)		//...	的len个字符	...
string s(cstr)			//以C-string构造
string(chars,chars_len)		//以C-string chars的前chars_len个字符构造

string s(begin,end)		//以迭代器区间构造
