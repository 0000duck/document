assert 斷言 test

assert 斷言 

#include <boost/assert.hpp>

assert 組件 提供了 多個 有用的宏

BOOST_ASSERT(expr)
在 debug 模式下  判斷 expr 如果斷言失敗 將 打印錯誤並 結束程序

如果定義了 BOOST_DISABLE_ASSERTS 則 BOOST_ASSERT 宏將被禁用

如果定義了 BOOST_ENABLE_ASSERT_HANDLER 則無論是否debug 
BOOST_ASSERT宏 都將生效
並且將 調用 boost namespace 下的 void assertion_failed(const char * expr,const char* function,const char * file, long line) 函數
	boost 故意沒有實現 assertion_failed 函數
	讓使用者 去實現 以便 可以安自己的方式 處理錯誤

example BOOST_ASSERT 

#include <iostream>

#define BOOST_ENABLE_ASSERT_HANDLER
#include <boost/assert.hpp>

namespace boost
{
    void assertion_failed(const char* expr,const char* function,const char* file, long line)
    {
        std::cout<<"error:"<<file<<"(at "<<line<<"line)\n";
    }
};
int main()
{
    BOOST_ASSERT(false);

    std::cout<<"Hello world!"<<std::endl;
    return 0;
}
