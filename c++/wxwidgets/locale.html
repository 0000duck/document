wxLocale 

wxWidgets 使用 wxLocale 提供對於 國際化的 支持
其支持 gettext 翻譯文件
_() 和 宏 中的內容 將會被 gettext 翻譯

一般在*app中定義一個 成員變量wxLocale
在 *app::OnInit 中 初始化 wxLocale

.mo文件 必須 放置在 相應文件夾下 如
正體中文 必須 放置在 zh_TW

example 

./locale/zh_TW/zh_TW.mo
bool *App::OnInit()
{
    if(locale_.Init(wxLANGUAGE_CHINESE_TAIWAN,wxLOCALE_DONT_LOAD_DEFAULT))
    {
        locale_.AddCatalogLookupPathPrefix(_T("."));
        if(locale_.AddCatalog("zh_TW"))
        {
            wxMessageBox(_("success"));
        }
        else
        {
            wxMessageBox(_("Couldn't find/load the catalog for locale."));
        }
    }
    else
    {
        wxMessageBox(_("This language is not supported by the system."));
    }
    ...
}
