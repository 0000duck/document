OpenCV 

OpenCV(Open Source Computer Vision Library) 是一個 開源(BSD) 跨平台 的計算機視覺 庫

可用於 人機交互 物體識別 圖像分區 人臉識別 動作識別 運動跟踪 機器人

官網http://opencvlibrary.sourceforge.net/
http://opencv.org/

http://www.opencv.org.cn/ 是一個 OpenCV 中文 bbs

core模塊 

OpenCV的基本數據結構 包括基本圖像的像素讀寫 及相關操作

imgproc模塊 

imgproc 包含了 常用的 圖像處理 (如 模糊 腐蝕...)

環境配置win32 

1	從官網下載對對應平台 api *.exe
2	win32已經提供 編譯好的lib dll
	設置 include lib bin 路徑到編譯環境	    
3	在新項目中 包含必要 lib 頭文件(248 為版本號)
#ifdef _DEBUG
#pragma comment(lib,"opencv_core248d.lib")
#pragma comment(lib,"opencv_imgproc248d.lib")
#pragma comment(lib,"opencv_highgui248d.lib")
#pragma comment(lib,"opencv_ml248d.lib")
#pragma comment(lib,"opencv_video248d.lib")
#else
#pragma comment(lib,"opencv_core248.lib")
#pragma comment(lib,"opencv_imgproc248.lib")
#pragma comment(lib,"opencv_highgui248.lib")
#pragma comment(lib,"opencv_ml248.lib")
#pragma comment(lib,"opencv_video248.lib")
#endif

#include<opencv2\imgproc\imgproc.hpp>
#include<opencv2\core\core.hpp>
#include<opencv2\highgui\highgui.hpp>
