blender 界面操作

blender 

一個開源(GPL) 跨平台的 三維繪圖及渲染軟體

官網 http://www.blendercn.org/

界面操作 

blender 使用 右鍵 點選 
使用 左鍵 移動準心(shift + c 移回準心到正中)
使用 鍵盤a 解除選擇/選擇全部
鼠標滾動 放大縮小
鼠標中鍵 旋轉畫面
(先)shift + 鼠標中鍵 移動畫面

視窗 

blender 默認只有一個 視窗
左擊 右上角 不放可以 拖來出一個新 刪除
右擊之 可以 合併視窗
