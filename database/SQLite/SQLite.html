SQLite 環境配置 常用命名

SQLite 

SQLite 一個 快速 開源 的嵌入式 關係型 數據庫

官網http://www.sqlite.org

環境配置 

1	從官網下載 最新 源碼(sqlite-amalgamation-3080702.zip 或者 
	sqlite-autoconf-3080702.tar.gz [附帶等makefile文件]) 

	api文檔(sqlite-doc-3080702.zip)

2	解壓 sqlite-autoconf-3080702.tar.gz 並編譯 
	.configure
	make
	
	在 .libs/ 文件夾下得到
	libsqlite3.a (靜態庫)
	libsqlite3.dll.a libsqlite3-0.dll(動態庫)
	libsqlite3.a (客戶端程序)

3	將 sqlite3.h 加入到 include 路徑

常用命名 

//創建 或 打開 數據庫 
sqlite3.exe test.db

//查詢存在的表		
select name from sqlite_master where type = 'table' order by name;
.table

//查詢表詳細信息	
select * from sqlite_master where type = 'table' order by name;  

//查詢表創建命令	
.schema table_name   

//獲取表字段		
PRAGMA table_info(table_name)
    
//設置顯示分隔符	    
.separator "-"

//顯示數據庫名	    
.databases

//退出	    
.quit

//修改表名    
alter table 舊表名 rename to 新表名

//增加列 
alter table 表名 add column 列名 數據類型 限定符

//開始/提交/取消 事務
BEGIN TRANSACTION
COMMIT
ROLLBACK

轉義字符 

	/   ->    //
	'   ->    ''
	[   ->    /[
	]   ->    /]
	%   ->    /%
	&   ->    /&
	_   ->    /_
	(   ->    /(
	)   ->    /)

like 

%	任意字符串
_	單個字符
[]	指定範圍
[^]	排除範圍
