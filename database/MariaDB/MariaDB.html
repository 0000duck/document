MariaDB 

MariaDB 是MySQL創始人 Michael Widenius 在Oracle收購SUN(此時MySQL被 SUN收購)後 開發的社區 維護版MySQL分支
其語法API完全 兼容 MySQL 故事有MySql的連接接口 可用在 MariaDB
且 文件名 配置完全同MySQL 故可直接 替代 MySQL 

官網https://mariadb.org/

MariaDB 

vi /etc/yum.repos.d/MariaDB.repo

[mariadb]
name = MariaDB
baseurl = http://yum.mariadb.org/5.5/centos5-x86
gpgkey=https://yum.mariadb.org/RPM-GPG-KEY-MariaDB
gpgcheck=1

yum install MariaDB-server MariaDB-client

cp /usr/share/mysql/my-small.cnf  /etc/my.cnf
