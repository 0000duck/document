作用语句
显示存在的数据库show databases;
创建数据库create database 数据库名
[DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci];
删除数据库drop database 数据库名;
设置数据库字符编码
于my.cnf中设置 [mysqld]和[client]选项default-character-set=utf8


数据库引擎
查看可用引擎 show variables like 'have%'; show engines;
引擎名特点
MyISAM速度快占用空间小但不支持事务和回滚
InnoDB支持事务和回滚
MEMORY数据库保存在内存中故速度快但不安全
其他非余
常用故未
列出
设置默认存储引擎
于my.cnf中设置 [mysqld]选项default-storage-engine=innodb（innodb是引擎名）
