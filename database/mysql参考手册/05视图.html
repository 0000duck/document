创建视图
create [undefined|merge|temptable] view 视图名 [(字段...)] as select ...[with [cascaded|local] check option];

with check option表示更新视图时要保证在该视图的权限范围之内
cascaded表示更新时要满足所有相关视图和表条件
local表示更新时只需要保证满足视图本身定义条件

undefined默认 具体定义余未了解
merge余暂时不解
temptable将视图结果存储到临时表，在再临时表中执行sql语句


查看视图
desc 视图名
show create view 视图名
show table status like '视图名'

mysql中视图信息存储在information_schema.views中
so select * from information_schema.views;


修改视图
alter [undefined|merge|temptable] view 视图名 [(字段...)] as select ...[with [cascaded|local] check option];
create or replace [undefined|merge|temptable] view 视图名 [(字段...)] as select ...[with [cascaded|local] check option];//若view不存在创建之
