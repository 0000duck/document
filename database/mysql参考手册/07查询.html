分组显示出字段所有信息 select group_concat(字段) from 表名 group by 字段
统计分组包含数据条数 select *,count(*) from 表名 group by 字段

order by 字段 asc|desc

消除重复 select distinct * from 表名

连接

内连接
select * from test1,test2 where test1.id=test2.id;

外连接
select 字段 from 表1 left|right join 表2 on 表1.字段名=表2.字段名
left显示表1全部内容表2符合要求才显示
right显示表2全部内容表1符合要求才显示


其他查询
select * from 表名 where 字段 in (select * from 表名);
exists 表示是否存在 select * from test where exists (select * from test2 );
any 表示满足其中之一 select * from test where id>=any (select id from test2);
all 表示满足其中全部 select * from test where id>=all (select id from test2);
select * from test where name regexp 'perl正则表达式'

合并
union 合并去除重复 union all 合并不去除重复
