创建表
create table name(id int primary key auto_increment,test int not null unique default 123)engine=innodb;
create table name(id int,id1 int,primary key(id,id1));
外键 constraint 外键别名 foreign key (属性1,属性2,...) references 表名(属性1,属性2,...);
查看表结构
desc 表名
show create table 表名

修改表
作用语法
修改表名alter table 旧表名 rename 新表名
修改字段名alter table 表名 change 旧字段名 新字段名 新数据类型
修改字段数据类型alter table 表名 modify 字段名 数据类型
修改完整约束alter table 表名 modify 字段名 数据类型 完整约束
修改字段位置alter table 表名 modify 字段名 数据类型 after 字段名;/(first)
删除字段alter table 表名 drop 字段名
增加字段alter table 表名 add 字段 数据类型 完整约束 ;/first /after 字段
删除外键约束alter table 表名 drop foreign key 外键别名
更改存储引擎alter table 表名 engine=存储引擎名

删除表
drop table 表名
若待删除的表是其他表的父表 需先删除外键约束 或子表
