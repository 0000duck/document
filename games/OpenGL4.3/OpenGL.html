OpenGl 環境配置 mingw example

OpenGl 

OpenGL 一個開源 高效 跨平台的 圖像接口 被廣泛用於 遊戲 圖像處理
官網 http://www.opengl.org/

mingw 

環境配置 mingw

通常 opengl的 環境不需要配置
只需要為連接器 加上 -lopengl32 選項(libopengl32.a) 即可

glut 

opengl 未提供任何 和窗口相關的 api
glut 是一個 輔助 工具包 提供了簡單的 窗口功能

下載 將 glut32.lib glut32.dll glut.h 加入即可

example 

#include <iostream>
#include <stdio.h>

using namespace std;

#include <gl/gl.h>
#include <gl/glu.h>
#include "glut.h"

void render_scene()
{
	//清除緩衝區 圖像(使用 背景色 填充)
	glClear(GL_COLOR_BUFFER_BIT);

	//設置繪製 填充顏色
	glColor3f(1.0f,0.0f,0.0f);

	//繪製矩形
	glRectf(50,100,100,50);

	//將內存繪製 交換到 窗口(會自動調用 glFlush)
	glutSwapBuffers();
}
int main(int argc,char** argv)
{
	//設置 窗口顯示 模式 GLUT_SINGLE(單緩衝) GLUT_RGB(RGB顏色模式)
	glutInitDisplayMode(GLUT_RGB | GLUT_SINGLE);


	//創建 窗口
	glutCreateWindow("hellow word");

	//返回 版本
	cout << glGetString(GL_VERSION) << endl;



	//設置顯示回調函數
	glutDisplayFunc(render_scene);

	//初始化 OpenGL 顏色分量取值 [0.0 1.0]
	glClearColor(1.0f,0.0f,0.0f,0.0f);	//設置背景清除顏色
	//...其他初始化操作

	//運行 glut 框架 直到結束程序
	glutMainLoop();


	return 0;
}
