cocos2d-js 環境配置 常用命令

cocos2d-js 

cocos2d-js 就是 cocos2d-x 的html5版本 

官網 http://cocos2d-x.org

環境配置 

1	安裝 java jdk 1.6+
2	安裝 python 2.7.x
3	如果需要打包到 android 先配置 android 環境
	從http://developer.android.com/sdk/index.html 下載安裝 sdk ndk
	從http://ant.apache.org/ 下載安裝 ant
	配置環境變量 
4	從 cocos2d-js 官網下載 最新 js 版 源碼 解壓 
	並運行 install setup.py 安裝

常用命令 

//運行以及創建的項目
cocos run -p web

//創建一個同時包含 Cocos2d-html5和Cocos2d-x JSB的項目
cocos new -l js

//創建僅包含Cocos2d-html5的項目
cocos new -l js --no-native

//同上 不過指定了工程名 和 位置
cocos new projectName -l js -d ./Projects

//編譯打包到 android apk
cocos compile -p android
