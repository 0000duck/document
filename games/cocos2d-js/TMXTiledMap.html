TMXTiledMap 磚塊地圖 

Class
cc.TMXTiledMap

Defined in: CCTMXTiledMap.js
Extends cc.Node

用處 

TMXTiledMap 對磚塊地圖 進行了 支持

屬性 

//地圖 寬/高 (磚塊數量 而非 像素等)
mapWidth
mapHeight

//每塊磚的 寬/高 (單位:像素)
tileWidth
tileHeight

//地圖定位(90度俯視 還是斜45度角 或6邊形)
mapOrientation

//一個 map 保存了所有地圖屬性
properties

//對象層數組
objectGroups

方法 

//構造函數 創建 地圖

//創建 地圖 使實例化和加載地圖 可以分開
initWithTMXFile(tmxFile)

//返回/設置 地圖尺寸(尺寸是磚塊數量 而非像素) (obj.width obj.height)
getMapSize()

	{cc.Size} Var

//根據名稱 返回圖層 TMXLayer
getLayer(layerName)

//返回/設置 objectGroups
getObjectGroups()
setObjectGroups(Var)
	Parameters:
	{Array} Var

//返回properties
getProperties()

getProperty(propertyName)
setProperties(Var)

getTileSize()

getMapOrientation()

	{Number} Var

example 

官方示例
1.//create a TMXTiledMap with file name
var tmxTiledMap = new cc.TMXTiledMap("res/orthogonal-test1.tmx");
2.
//create a TMXTiledMap with content string and resource path
var resources = "res/TileMaps";
var filePath = "res/TileMaps/orthogonal-test1.tmx";
var xmlStr = cc.loader.getRes(filePath);
var tmxTiledMap = new cc.TMXTiledMap(xmlStr, resources);

實例對象 和加載地圖分開
var map = new cc.TMXTiledMap();
var ok = map.initWithTMXFile("res/a.tmx");
if(ok){
	this.addChild(map, 0, TAG_TILE_MAP);
}
