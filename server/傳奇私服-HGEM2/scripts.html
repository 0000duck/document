代碼段 

[@段名] 定義一個代碼段
#ACT	後面代碼 作為腳本 執行
		GOTO @段名 執行代碼段
#SAY	... 显示界面
			使用 <顯示文本/@段名> 執行代碼段
			使用 \換行


BREAK	act 結束標記 (其實似乎沒什麼用)
CLOSE	後面的代碼依然會執行 關閉對話框

調用子腳本 

//路徑 是以Mir200\Envir\QuestDiary 為當前路徑
#Call [路徑] 段名

被調用的 段 必須使用 {} 擴起來

example 

[@callmob]
#ACT
	#CALL [\dark\clock.txt] @test
	#CALL [\dark\clock.txt] @test2


QuestDiary\dark\clock.txt
[@test]
{
#ACT
	GoTo @show

[@show]
#SAY
	yes
}

[@test2]
{
#SAY
	2
}

系統變量 

P0-P9 私人变量(数字型) 不可保存
G0-G99 全局变量(数字型) 可保存
M0-M99 私人变量(数字型) 不可保存
I0-I99 全局变量(数字型) 不可保存
D0-D9 私人变量(数字型) 不可保存
N0-N99 私人变量(数字型) 不可保存
S0-S99 私人变量(字符型) 不可保存
A0-A999 全局变量(字符型) 可保存 

注意：自定义变量不要以P、G、M、I、D、N、S、A开头

操作 

使用 類似 彙編的方式 操作(自定義變量 也可這樣 操作)
//l = r
MOV l r

//l -= r
DEC l r

//l += r
INC l r
SUM l r

//v = l / r
DIV v l r

//v = l * r
MUL v l r

//v = l / r * 100
PERCENT v l r



//判斷 l<r 是否為真
SMALL l r

//判斷 l>r 是否為真
LARGE l r

//判斷 字符串 是否相等(對於字符串 比較 不區分大小寫)
EQUAL 變量 字符串

注意 

EQUAL 操作 的右值 不能是 變量

SMALL LARGE 如果 右值 為 變量 其可能存在 bug 
(會將 值作為字符串比較 即使 是 數字)
