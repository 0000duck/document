easydarwin 

easydarwin 是一個開源的(APPLE PUBLIC SOURCE LICENSE Version 2.0 ) 流媒體 平台 框架

EasyDarwin 提供了 RTSP/HLS/HTTP 協議的支持
提供了 點播(Mp4) 直播(H264/MJPEG/MPEG4、AAC/PCMA/PCMU/G726) 

官網 http://www.easydarwin.org/

點播 

1	下載最新 easydarwin源碼 並編譯
	git clone https://github.com/EasyDarwin/EasyDarwin.git
	(easydarwin 為win32 提供了 編譯好的 程序
	https://github.com/EasyDarwin/EasyDarwin/releases

	https://drive.google.com/file/d/0BzBTagdHxViwVDBPakVnTnhPN2s/view?usp=sharing
	)


2	修改配置文件 easydarwin.xml
	rtsp_port	rtsp服務端口
	movie_folder	流媒體文件目錄

3	運行 EasyDarwin.exe -c ./easydarwin.xml -d

4	使用 rtsp://[ip]:[rtsp_port]/[filename] 訪問 視頻流

注意 

rtsp 服務器只 處理 帶有 rtsp信息頭的 mp4
需要 為 mp4 添加 信息頭

mp4box a.mp4 -hint
