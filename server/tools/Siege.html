Siege 

Siege 一個開源的 web 壓力測試 工具

官網 https://www.joedog.org/



//對 url.txt 中的地址(每行一個) 隨機訪問 100次 每次 並發 300個訪問
siege -c 300 -r 100 -f url.txt
	-C	顯示 當前 配置
	-v	運行時 顯示 詳細信息
	-c	並發量
	-i	隨機訪問
	-d	每個 uri之間的 延遲
	-r	重逢次數 不能 與 -t 一起使用
	-t	持續時間 單位分 (3600S = 60M = 1H) 不能和 -r 一起使用
	-l	指定 運行結束 日誌 保存 位置 
	-R	指定 配置文件 位置siege
	-f	多個url地址 文件	
	-u	測試url
