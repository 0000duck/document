触发功能 

E:\MirServer\Mir200\Envir\Market_Def\QFunction-0.txt 中定義 功能 代碼 如 觸發指令

調用子腳本 

#CALL [\path] @code
(調用子 腳本
#CALL [\装备回收.txt] @装备回收)

path 為子腳本相對E:\MirServer\Mir200\Envir\QuestDiary路徑
@code 為被調用 程序段
(被調用程序段 必須 以 {} 擴起來)

example 

;MirServer\Mir200\Envir\Market_Def\QFunction-0.txt
;此脚为功能脚本，用于实现各种与脚本有关的功能
;人物升级触发
[@PlayLevelUp]
#ACT
	#CALL	[back30.txt] @main


;MirServer\Mir200\Envir\QuestDiary\back30.txt
[@main]
{
#IF
	CheckMapName	SELF	G008
#ACT
	GOTO @lv20	
BREAK


[@lv20]
#IF
	CHECKLEVELEX > 30
#ACT
	mapmove	3	330	330
BREAK

}
