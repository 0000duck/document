#ELSEACT

[@测试2]

	CHECKLEVELEX = 40

腳本語法 

以	[@*]	定義一個	程序段([@main] 為入口點)

以	</>	定義一個 用戶選擇的交互接口 
以	/	隔開 顯示的選擇名 和 觸發命令(程序段) 

以 IF BREAK 	定義一個 判斷(IF 後接條件)
以 ACT		後接 true 時 執行命令
以 ELSEACT	後接 false 時 執行命令

以 GOTO [@*]	將出現執行 跳轉到指定程序段

以 \回車	 為顯示換行

注意 

此腳本語法 充滿了 bug
在 break 之後的 語句 將會被 if為真 執行 或被 else執行 
其 被 歸到 最近的 一個 if 或 else 程序塊中
(不建議 在break後 寫代碼)

IF之 處理 ACT ELSEACT 執行命令外
亦可 SAY ELSESAY 顯示 文本 然此語法 充滿 bug(不建議 使用)
可以選擇 直接 GOTO 到一個 新程序段 顯示 如example 的 [@test]

物品買賣腳本 

買賣腳本 語法同 普通腳本
不同 其必須在開頭 標記 支持的 特殊段（操作） 
一些程序段 必須用指定名 不如 買 必須使用 sell 賣物品
還要在 開頭 以 % 指定物價 調整(%200 以200%的物價)
以 + 數字 指定 支持的操作
在結尾 以 [goods] 指定商店物品

特殊段 

@s_repair 特殊修理
@buy 购买东西
@sell 出售东西
@repair 普通修理
@makedrug 制作毒药
@storage 保管物品
@getback 取回物品
@exit 关闭窗口
@upgradenow 升级武器
@getbackupgnow 取回武器

+數字 

+40 买肉 
+1 卖肉 
+43 武器有关(修) 
+15 头盔 
+10 买衣服 
+11 修衣服 
+26 修手镯类 
+0 买各种药水 
+42 卖各种药水 
+30 蜡烛等.
+25 特殊水
+3 买卷轴,书类 
+4 卖书类 
+19 项链 
+20 项链 
+21 项链 
+22 买戒指 
+23 卖戒指 
+24 买手镯 
+26 卖手镯 
+5 修炼武器或买武器 
+6 修炼武器或卖武器
